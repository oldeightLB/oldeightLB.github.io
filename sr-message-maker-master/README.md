## 功能

- [x] 自动播放
- [x] 绿幕背景
- [x] 男女主切换
- [x] 自定义头像
- [x] 自定义角色
- [x] 表情包
- [ ] 自定义表情包
- [x] 通知信息
- [x] 图片消息
- [x] 图片剪裁
- [x] 任务消息
- [x] 回复选项
- [x] 对话气泡
- [x] 数据导出导入
